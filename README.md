# GFModules documentation repository

## Prerequisites

- Node.js and npm
- Make
- Docker (with Sphinx and Python)

## Setup

1. Install project dependencies:

```bash
npm install
```

## Development

Compile and generate the documentation locally by running:

```bash
make all
```

Start the local (live)html server with auto-reload:

```bash
make livehtml
```
